# Desktop Task Manager

##  Описание проекта

**Desktop Task Manager** — это настольное приложение для управления личными задачами.  
Проект разработан в учебных целях для демонстрации организации репозитория, управления задачами (Issues), работы с Kanban-доской и применения базовых принципов гибкой методологии разработки.

Приложение позволяет пользователю создавать, редактировать и отслеживать выполнение задач в удобном графическом интерфейсе.

---

##  Цель проекта

- Освоить создание и структуру репозитория
- Научиться работать с Issues и метками
- Использовать Kanban-доску для управления задачами
- Реализовать базовое десктопное приложение

---

##  Функционал

Приложение поддерживает следующие возможности:

-  Создание новых задач  
-  Редактирование существующих задач  
-  Удаление задач  
-  Изменение статуса задачи (Новая / В работе / Выполнена)  
-  Фильтрация задач по приоритету и статусу  
-  Сохранение задач в локальный файл  
-  Автоматическая загрузка данных при запуске

---

##  Пример использования

1. Запустить приложение  
2. Нажать кнопку «Добавить задачу»  
3. Ввести название и описание  
4. Выбрать приоритет  
5. Сохранить задачу  

Задача отобразится в общем списке.

---

##  Методология разработки

Проект разрабатывается с использованием принципов гибкой методологии:

- Управление задачами через Issues  
- Использование меток (Labels)  
- Визуализация работы через Kanban-доску  
- Разделение задач по статусам
